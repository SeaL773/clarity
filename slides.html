3-Step AI Pipeline

Not a single API call — three dedicated prompts, each designed for its task.

1. Parse → Extract Tasks

Finds every task — even implicit ones. "I need to eat healthier" → "Plan meals" + "Go grocery shopping"

2. Plan → Prioritize & Order

Eisenhower Matrix priority. Quick wins first for momentum. Workflow-aware ordering.

3. Summarize → Daily Recap

Encouraging summary. Leads with wins, never guilt. Suggests tomorrow's focus.
